# rest-with-spring-boot

Projeto criado para documentar e praticar o estudo de Java com Spring Boot

### Repositorio em desenvolvimento ...

## Fundamentos trabalhados no decorrer do estudo

- Fundamentos do REST
- Path Params e Exceptions
- Verbos Http
- Spring Data JPA e integração ao MySql
- Logs
- DTO
- Versionamento
- Migrations com Flyway
- Custom Json Serializattion
- Content Negotiation
- HATEOAS
- Swagger
- Docker
- TestContainers e REST Assured
- Cross-Origin Resource Sharing (CORS)
- Query Params e Paginação
- Upload e Download de arquivos
- Importação e Exportação de arquivos Xlsx e Csv
